liangchen提示:为了获得最佳游戏体验,请牢记以下目前还没有过时的游玩技巧:

  • 创建一个队伍,队长就像是"主机"
  • 只有队长可以与任务npc对话来推进任务。
  • 只有队长才能去拾取关键任务物品(龙石啥的)
  • 建议在这场奇妙的联机之旅中,每次都保证你的队长是同一个人(不然就得同步任务进度再组队)
  • 如果人少,建议服务器上只有一只队伍
  • 等队长先进地图,队员后进,保证怪物是从队长同步的
